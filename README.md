## Usage

|               | GET           | PUT  |POST | DELETE |
| ------------- |:-------------:| -----:| -----:| -----:|
| records       | retrieve all payment records |   | | |
| record/:id    | retrieve payment record with record_id = :id |  | | |
| profiles/     | retrieve all user profiles   |     | | |
| profile/:id   | retrieve user profile with record_id = :id   |    | | |
